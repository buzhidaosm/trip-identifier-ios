# Code Samples

These Swift files are intentionally trimmed portfolio excerpts. They demonstrate
implementation patterns from the iOS app without publishing the full production
source code or proprietary recommendation logic.

- `VisionOCRSample.swift` shows on-device text recognition with Apple Vision.
- `LiveActivitySample.swift` shows ActivityKit / Dynamic Island presentation.
- `SwiftUIDashboardSample.swift` shows SwiftUI state-driven UI composition.

The complete production app, trip parsing rules, net-value calculations, and
offer-scoring algorithms are not included in this public repository.
